Bathyspheres
  • During my time making music professionally I realized that having cloud based electronic music software would be beneficial.
  • Bathyspheres allows a user to login, load up a project they’ve shared with a collaborator, and get right to making music. One user might be working on writing the music while another simultaneously edits the synth to get that perfect sound.
  • Technologies used are PostgreSQL, JavaScript, AngularJS, AngularJS Material, Web Audio Api, Node, HTML, and CSS.

Hustle
  • There is no way for skilled people to connect and work on side projects together.
  • Hustle allows users to log-in, connect with other people to collaborate on projects together. While working on a project together users can track project’s progress and communicate through Hustle. Finally users can rate how well the project and collaborators worked.
  • Technologies used are JavaScript, AngularJS, AngularJS Material, Node, HTML, CSS, OAuth2.0 for Facebook and Google, NodeMailer.
